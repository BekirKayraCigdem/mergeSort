# Merge Sort

------------------------------------

[16,21,11,8,12,22] -> Merge Sort <br>

Yukarıdaki dizinin sort türüne göre aşamalarını yazınız. <br>

[16,21,11,8,12,22]

*       [16,21,11,8,12,22]
*       /             \
*   [16,21,11]       [8,12,22]
+      /     \          /     \ 
+  [16,21]   [11]    [8,12]  [22]
+   /   \      |      /  \     |
+ [16] [21]   [11]  [8] [12]  [22] 
+   \  /      |      \  /     |
+  [16,21]    [11]    [8,12]  [22]
+      \      /         \     /
+     [11,16,21]       [8,12,22]

Big-O  

O(nlogn)
