# Merge Sort

------------------------------------

[16,21,11,8,12,22] -> Merge Sort <br>

        /             \<br>
   [16,21,11]       [8,12,22]<br>

 [16,21]   [11]    [8,12]  [22]<br>

[16] [21]   [11]  [8] [12]  [22] <br>
